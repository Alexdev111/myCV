# Personal CV Website

## Description

This project is a personal CV website developed as part of the Introductory Practice in the field of Informatics.

The website presents personal information, education, technical skills, work experience, projects, and contact details in a modern and responsive format.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Features

- Responsive design
- Dark / Light mode
- Personal profile section
- Education section
- Experience section
- Projects section
- Skills section
- Contact information
